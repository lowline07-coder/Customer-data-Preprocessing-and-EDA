# Customer-data-Preprocessing-and-EDA
A complete Data Science project demonstrating Data Acquisition, Data Cleaning, Data Profiling, and Exploratory Data Analysis (EDA) using customer purchase behavior data with Python, Pandas, NumPy, Matplotlib, and Seaborn.
# Customer Data Preprocessing and EDA

## Project Overview

This project demonstrates the complete workflow of:
- Data Acquisition
- Data Cleaning
- Data Profiling
- Exploratory Data Analysis (EDA)

using customer purchase behavior data.

The project was developed using Python and popular Data Science libraries such as:
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite

---

# Objective

The main objective of this project is to:
- Understand and preprocess raw data
- Perform data cleaning
- Analyze customer behavior
- Generate insights using EDA
- Create data profiling reports

---

# Dataset Information

The dataset contains customer purchase behavior information with the following columns:

| Column | Description |
|---|---|
| Customer_ID | Unique customer identifier |
| Age | Customer age |
| Income | Customer income |
| Purchases | Number of purchases |
| Gender | Male/Female |
| Churn | Customer churn status |

# How to Run the Project

1. Clone the repository

```bash

```

2. Open the project folder

3. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Open Jupyter Notebook

```bash
jupyter notebook
```

5. Run all cells in:
- project1.ipynb
